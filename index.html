Veritabanı bağlantısı

Tüm cihazlarda senkronize çalışması için ücretsiz Supabase hesabına bağla. Bir kez kurulunca her cihazda aynı liste görünür.

1
supabase.com'a git, ücretsiz hesap aç ve yeni bir proje oluştur.
2
Sol menüden SQL Editor'ı aç ve aşağıdaki kodu çalıştır:

create table activities ( id bigint primary key generated always as identity, title text not null, cat text default 'Doğa', date text, done boolean default false, photos jsonb default '[]', sort_order int default 0 ); alter table activities enable row level security; create policy "public access" on activities for all using (true) with check (true);
3
Project Settings → API sayfasından URL ve anon public key'i kopyala ve aşağıya yapıştır.
